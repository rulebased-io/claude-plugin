# /score — Harness Score Report

Generate a detailed per-category score report showing how well each harness area is implemented.

## Usage

```
/rulebased-harness:score
/rulebased-harness:score ./path/to/project
```

Or via CLI:
```bash
npx @rulebased/cli score
```
