Create today's daily note at `daily/YYYY-MM-DD.md` and run the daily brain routine.

Morning: show inbox count, surface 2-3 review notes, carry forward yesterday's unfinished tasks, show recent captures.
Evening: process inbox, summarize captures, check task completion, prompt reflection.

One daily note per day. Use `templates/daily.md` if it exists.
