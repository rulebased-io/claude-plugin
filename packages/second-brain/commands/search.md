Search across all notes using multiple strategies.

Query syntax: plain text (full-text), `#tag`, `@YYYY-MM-DD` (date), `in:folder`, `status:X`, `orphan`. Combine filters with AND.
Reads AGENTS.md for folder structure. Results ranked by relevance (max 10), with context snippets.
Case-insensitive. Archived notes included but marked `[archived]`.
