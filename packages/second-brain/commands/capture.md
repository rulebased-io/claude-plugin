Rapidly save an idea, reference, or thought into `inbox/` before it's lost.

1. Extract or ask for content (free-form text, URL, quote, code snippet)
2. Generate filename: `inbox/YYYY-MM-DD-<slug>.md` (kebab-case)
3. Create note with frontmatter per `system/conventions.md` (`title`, `created`, `tags`, `status: seedling`)
4. Auto-tag: up to 3 tags based on content
5. Confirm with file path and summary
