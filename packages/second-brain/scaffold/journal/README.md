# Journal

Daily reflection entries. One file per day.

## File naming

`YYYY-MM-DD.md` (e.g., `2026-03-19.md`)

## Guidelines

- Use `templates/journal.md` for consistent structure
- Keep entries honest and concise
- Link to relevant notes with `[[wikilinks]]`
- Optional: add `mood` and `energy` fields to frontmatter

## When to reorganize

When entries exceed 3 months, consider grouping by month:

```
journal/
├── 2026-01/
│   ├── 2026-01-01.md
│   └── 2026-01-02.md
└── 2026-03/
    └── 2026-03-19.md
```

See `system/growth-triggers.md` for the specific threshold.
