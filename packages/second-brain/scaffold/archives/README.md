# Archives

Completed projects, former areas of responsibility, and outdated resources.

## What gets archived

- Completed projects (move entire folder here)
- Areas you're no longer responsible for
- Resources that are no longer relevant
- Old notes superseded by newer ones

## Guidelines

- Archive, never delete — you may need it later
- Archived content is still searchable and linkable
- Do not organize archives — just move items in
- Review annually to clean up truly obsolete items
