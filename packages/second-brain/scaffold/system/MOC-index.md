## MOC 생성 가이드

**MOC를 만들 때**:
- 특정 주제의 관련 항목이 3개 이상 모였을 때
- 반복적으로 같은 파일들을 찾아다닐 때
- 프로젝트가 시작되어 관련 자료를 모아야 할 때

**MOC 형식**:
```markdown
---
title: [주제] MOC
date: YYYY-MM-DD
type: permanent
tags: [moc, 주제태그]
---

# [주제] MOC

> 한 줄 설명

## 핵심 항목
- [[항목1]] — 간단한 설명
- [[항목2]] — 간단한 설명

## 관련 주제
- [[다른-MOC]]
```

**원칙**:
- MOC는 상향식으로 만든다 — 콘텐츠가 먼저, 구조는 나중
- 빈 MOC를 미리 만들지 않는다
- MOC끼리도 연결한다 (메타 구조)
