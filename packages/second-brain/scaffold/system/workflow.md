# Workflow (CODE)

Knowledge processing flow based on Tiago Forte's CODE framework.

## 1. Capture

Drop it into `inbox/`. Don't worry about format, classification, or quality.

- Free text, URLs, quotes, code snippets — anything goes
- Minimum frontmatter: `title`, `created`, `tags`
- File naming: `YYYY-MM-DD-slug.md`

## 2. Organize

Clear inbox within 7 days. Move each note to the right place:

```
inbox/ arrival
    ↓
  Classify:
    Project-related?       → projects/<name>/
    Ongoing area?          → areas/
    Reference / learning?  → resources/
    Daily reflection?      → journal/
    Not useful?            → delete
```

## 3. Distill

Extract and refine the core ideas:

- **Bold** the key insights
- Add a summary section
- Promote `status` from `seedling` → `budding` → `evergreen`
- Connect to related notes with `[[wikilinks]]`

## 4. Express

Turn accumulated knowledge into output:

- Blog posts, technical docs, decisions
- Synthesize multiple notes into new insights
- Apply directly to projects
