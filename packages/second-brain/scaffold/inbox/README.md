# Inbox

Drop anything here. Don't worry about format, tags, or where it belongs.

## Capture → Classify → Move

```
inbox/ arrival
    ↓
  Classify:
    Active project work?   → projects/<name>/
    Ongoing responsibility? → areas/
    Reference / learning?  → resources/
    Daily reflection?      → journal/
    Not useful?            → delete
```

## Guidelines

- File naming: `YYYY-MM-DD-slug.md`
- Minimum frontmatter: `title`, `created`, `tags`
- Process inbox at least weekly — aim for inbox zero
- When in doubt, capture anyway. Organize later.
